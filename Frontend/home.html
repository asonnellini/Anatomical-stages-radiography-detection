Deep to the Bones


Hello and welcome!

Deep to the Bones is an App that can detect Anatomical Stages on your radiographies.

To run the detection:

  • 1) Click on Choose a file button
  • 2) Select a file (supported formats are jpg, jpeg, png - DICOM is coming soon!)
  • 3) Click on Run Detection button
  • 4) Click on Download the image below the image to download it

On the left menu you can find the link to our GitHub - Enjoy!
